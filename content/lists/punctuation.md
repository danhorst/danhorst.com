> Using the right punctuation is important.

| Character               | Literal  | Alphanumeric          | Unicode             |
|-------------------------|----------|-----------------------|---------------------|
| Ampersand               | &        | `&amp;`               | `&#38;`             |
| Bullet                  | •        | `&bull;`              | `&#8226;`           |
| Degree                  | °        | `&deg;`               | `&#176;`            |
| Double quotes           | “ ”      | `&ldquo;` `&rdquo;`   | `&#8220;` `&#8221;` |
| Single quotes           | ‘ ’      | `&lsquo;` `&rsquo;`   | `&#8216;` `&#8217;` |
| Ellipsis                | …        | `&hellip;`            | `&#8230;`           |
| Em dash                 | —        | `&mdash;`             | `&#8212;`           |
| En dash                 | –        | `&ndash;`             | `&#8211;`           |
| Lozenge                 | ◊        | `&loz;`               | `&#9674;`           |
| Middle dot              | ·        | `&middot;`            | `&#183;`            |
| Numero                  | №        | `&numero;`            | `&#8470;`           |
| Pilcrow                 | ¶        | `&para;`              | `&#182;`            |
| Plus/minus              | ±        | `&plusmn;`            | `&#177;`            |
| Section                 | §        | `&sect;`              | `&#167;`            |
| Single & double daggers | † ‡      | `&dagger;` `&Dagger;` | `&#8224;` `&#8225;` |
| Filled star             | ★        | `&starf;`             | `&#9733;`           |
| Star                    | ☆        | `&star;`              | `&#9734;`           |
| Fractional quarter      | ¼        | `&frac14;`            | `&#188;`            |
| Fractional third        | ⅓        | `&frac13;`            | `&#8531;`           |
| Fractional half         | ½        | `&frac12;`            | `&#188;`            |
| Fractional two thirds   | ⅔        | `&frac23;`            | `&#8532;`           |
| Fractional tree quarter | ¾        | `&frac34;`            | `&#190;`            |
| En space                |          | `&ensp;`              | `&#8194;`           |
| Em space                |          | `&emsp;`              | `&#8195;`           |
| Thin space              |          | `&thinsp;`            | `&#8201;`           |
| Hair space              |          | `&hairsp;`            | `&#8202;`           |
