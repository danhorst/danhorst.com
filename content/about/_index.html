Colophon

This website is designed to be cheap to run and fast to use. I use static site builder because it meets these goals and leaves me more time for writing and other things I like to do than a more complicated solution.

  • DNS: Hover
  • Hosting: Netlify (I migrated from GitHub pages so I could use Hugo. They also provide free HTTPS thanks to Let’s Encrypt.)
  • git Hosting: BitBucket
  • Site builder: Hugo
  • Typeface: ET Book
  • Site theme derrived from Hugo Tufte CSS which, in turn, is based on Tufte CSS
  • Service Worker configuration based on the work of Erik Runyon for nd.edu
  • Page-to-page navigation sped up by Turbolinks
  • Performance auditing via Lighthouse
