# danhorst.com

[Hugo][1] site that generates [danhorst.com][2].

New blog entry:

NOTE: Leave off the "content" directory at the beginning of the path. It is implicit.

```
hugo new writing/name_of_post.md
```

NOTE: Changes to the assets (CSS, JS, webfonts, etc.) require an update to the cache date in `sw.js`

[1]: https://gohugo.io/
[2]: https://www.danhorst.com
